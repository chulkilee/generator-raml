# generator-raml

> [Yeoman](http://yeoman.io) generator for [RESTful API Modeling Language (RAML)](http://raml.org/)


## Getting Started

```bash
# install Yeoman
npm install -g yo

# install it
npm install -g generator-raml

# run the generator
yo raml

# watch and serve
npm run-script serve
```
